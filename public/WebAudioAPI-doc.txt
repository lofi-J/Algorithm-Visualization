# Web Audio API 

1. AudioContext생성
    const audioContext = new (window.AudioContext || window.webkitAudioContext)();
* 크로스 브라우저 지원을 위해 단축평가 활용함

2. 소스Node생성
2-1: 메서드 방식
    const oscillator = audioContext.createOscillator();
2-2: 생성자 방식
    const oscillatorNode = new OscillatorNode(audioContext);
* 생성자 방식은 일부 브라우저에서 지원되지 않을 수 있음

3. 작업 Node생성 
일반적으로 볼륨 설정 역할을 함
3-1: 메서드 방식
    const gainNode = audioContext.createGain();
3-2: 생성자 방식
const gainNode = new GainNode(audioContext, {gain: 0.5});
* 생성자 방식은 일부 브라우저에서 지원되지 않을 수 있음

    gainNode.gain.value = 0.5; // 볼륨 설정(0~1사이)

4. Audio Graph 연결
: 소스Node ~ 작업Node ~ 목적지Node 연결
oscillatorNode.connect(gainNode).connect(audioContext.destination);

5. 재생
oscillatorNode.start();


# OscillatorNode로 다양한 음원 만들기
: defualt set => 440hz, sine
OscillatorNode 객체를 생성하여 start()메서드로 재생하면 디폴트로 주파수 440 Hz인 sine파의 음을 출력한다.

OscillatorNode가 출력하는 음의 높낮이와 파형은 개발자가 직접 변경할 수 있다.

3개의 프로퍼티와 1개의 메서드

1. frequency 프로퍼티
- Hz 설정: defualt 440hz
- 음정(높낮이)를 hz단위(주파수)로 조정하는 프로퍼티다.

2. detune 프로퍼티
- Cent 설정: defulat 0
- 음정(높낮이)를 cent단위로 조정하는 프로퍼티.

3. type 프로퍼티
- 파형 설정: "sine"(기본값), "square", "sawtooth", "triangle", "custom"
- 진동에 따른 공기 밀도의 변형 형태, 즉 파형을 선택할 수 있다.(파형에 따라 음색이 달라짐)

4. OscillatorNode.setPeriodicWave(PeriodicWave) 메서드
- 커스텀 파형 설정 메서드로 개발자가 원하는 파형을 가진 소리를 만들 수 있다.
